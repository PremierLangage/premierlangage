Behavior

Drag & Drop refers to both draggable and droppable components. The difference between the two is that a droppable component has the property droppable set to true

When a draggable component is dropped into a droppable component, the cid of the dragged one is affected to the property droppedId of the dropped one.

A draggable component can be dropped into multiple drop zones.

A drop zone can be cleared by dragging it into it's source.
