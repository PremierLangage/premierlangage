Representation

A MatchListItem is a JSON object with the following properties (? = optional)

{
    "id": string, // the id of the item
    "target": boolean, // a value indicating whether the item is a target (placed at the right).
    "source": boolean, // a value indicating whether the item is a source (placed at the left).
    "content": string, // HTML, markdown, MathJax or text content of the item.
    "multiple"?: boolean, // a value indicating whether the item can be connected multiple items.
    "css"?: string, // css classes of the item separated by a space.
}

A MatchListLink is a JSON object with the following properties (? = optional)

{
    "source": string, // id of the source item
    "target": string, // id of the target item
    "css"?: boolean, // css classes of the item separated by a space.
}
