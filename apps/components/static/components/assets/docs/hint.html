Representation

Hint is made up of multiple rows of items which can contain html, markdown, MathJax or text content. You specify the rows by initializing the property items of the component which is an array of HintItem objects. A HintItem object has the following properties: 

{
    "content": string, // HTML, markdown, MathJax or textual content of the item.
    "consumed": boolean, // A value indicating whether the hint is consumed by the student.
    "css": string, // optional css classes of the component separated by a space.
}

Dynamic Hinting

The default behavior of the component is not dynamic. The component will present the next non consumed hint of items array each time a student ask for a hint.

To create dynamic hints. The component provides the property script. The value of this property should be a javascript script. You can create a very complex and intelligent hinting mecanism using this property.

Combined with CountDown component, you can display hints to the student after a delay.
