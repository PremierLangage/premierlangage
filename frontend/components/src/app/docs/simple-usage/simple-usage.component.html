Simple Usage

The usage guide will based on the Sort List component of the introduction page. This component allow to sort a list of items by dragging them and we will see step by step how it works.

Each component documentation page comes with an interactive exercise like the following. You can modify the code inside the editor and click on the preview button to play with the example.

Usage explained

  • # Step 1: Declaration

    This block of code represents the declaration of a component. Here we use operator =: to tell the framework to creates new SortList component. After that we specify the property items of the component that will be used to render the items inside the exercise.

    In the first example we use the JSON syntax to provides the items of the component.

    In the second example we use a string syntax to provides the items of the component.

  • # Step 2: Randomization.

    Once created we have to randomize the items of the list and keep track to the answer inside a grader.

    In the first example we just put the id of the items inside a list answer and call random.shuffle() on the items to randomize the list.

    In the second example we use the predefined functions parse_string() and remind() of SortList class to do the same thing. The first one will split the string items using the specified separator \n in this case and create a JSON object like in the first example with random id for the items.

  • # Step 3: Exercise form

    From this block of code, we define the layout of the exercise and we insert the component in the HTML using the |component filter on the variable that hold the component. This filter will write the html code needed to render the component and map the properties of the component to the HTML element.

  • # Step 5 Evaluation

    After the student submit the form to the server to get a grade, the framework will sync the state of the component with the actions of the student. For example in the case of the sort list, When a student reorder the items the component will update the ordering of the property sortlist.items. So we compare the id of the elements on the list send by the student with the saved list answer and update the css property of the items to provide a feedback.

    In the case of the second example, we just call the method auto_grade() on the component to auto evaluate the exercise.
