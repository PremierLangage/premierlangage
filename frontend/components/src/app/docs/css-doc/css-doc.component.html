Component framework comes with a full featured API to custom the css of all the components. Everytime you see a property css on a component, the value of this property should be a space separated list of css classes.

Example:
sortlist.items[0]["css"] = "a b c" will add the css classes a, b and c to the items[0] of the Sort List component.

The framework provides 80+ css classes that you can use to customize the components. The classes are listed in the section below.

You can click on an animation of Animation section to apply it to the elements of the Appearance section. You can copy the css of the elements of Appearance section by clicking on an element.

  • delay-2s
  • delay-3s
  • delay-4s
  • delay-5s
  • infinite

Usage

If the Css API is not enough for your needs, you can create your own css classes and add them to the components.
