Representation

A checkbox group is represented by an array of CheckboxItem objects.

Each CheckboxItem has the following properties

{
    "id": string, // id of the item (required).
    "content": string, //  HTML, markdown, MathJax or text content of the item.
    "css": string, // optional css classes of the item separated by a space.
    "checked": boolean, // optional value indicating whether the item is checked (false if not specified).
}
