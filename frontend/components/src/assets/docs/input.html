Behavior

The behavior of the component depend on the value of the property type:

  • number:

    The text entered by a student is converted to an int or float before the grader is called. If the input box is empty then the value will be set to 0.

    In this mode the component ensures that the student cannot enter a non numeric value.

    Important:

    If a student enter the number 0.0 it will be converted to 0.

  • text:

    In this mode the logic of converting and handling the value entered by a student is delegated to the grader. You must cast the text to an int or a float and handle the case where the input box is empty by yourself.

  • multiline: Behaves like for when the property type is defined to text.
